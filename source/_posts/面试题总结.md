12. 常考：如何实现数组去重？

    ES6

    ```
    let arr = [1,2,1,3,2]
    [...new set(arr)]
    ```

    ES5

    ```javascript
    function unique(arr){
      for(let i = 0;i<arr.length;i++){
        for(let j = i+1;i<arr.length;i++){
          if(arr[i]===arr[j]){
            arr.splice(j,1);
            j--
          }
        }
      }
      return arr
    }
    ```

    使用includes

    ```javascript
    function unique(arr){
    	var newArr = []
      for(let i=0;i<arr.length;i++){
        if(!newArr.includes(arr[i])){
          newArr.push(arr[i])
        }
      }
      return newArr
    }
    ```

2. 英语题：loader 和 plugin 的区别是什么？

   loader:模块转换器，把对应的模块按照需求用对应的loader转换成新内容

   plugin:插件，webpack在构建时会在特定的时机广播出对应的事件，插件监听这些事件，做出对应的操作

3. 必考：如何按需加载代码？
